Starter HTML/CSS/JS files

Files created:
- `index.html` — basic HTML page linking `styles.css` and `script.js`.
- `styles.css` — minimal, responsive starter styles.
- `script.js` — DOM ready handler and example click event.

Quick preview

From the project folder run a simple static server, e.g. with Python 3:

```bash
python3 -m http.server 8000
```

Then open http://localhost:8000 in your browser.
